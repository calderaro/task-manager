Task Manager
===============

## Installation

### Clone
```bash
git clone git@github.com:calderaro/task-manager.git
```
### Install dependencies
```bash
npm install
```
### Run build production
```bash
 npm run build-production
```
### Run production server
```bash
 npm run start-production
```
### Run development server
```bash
 node index
```
